The following commands access the built-in documentation:

 

man (optional section number) <command or topic> - displays the system manual page about that command or topic

 

info <command or topic> - displays in-depth information (if available) about that command or topic

 

apropos <command or topic> - displays a list of commands and topics related to the one you are interested in

 

whatis <command or topic> - displays a list of man pages with the same name as the command or topic you specified

 

which <command> - displays a list of all commands in your current path with the same name
